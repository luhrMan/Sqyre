# Product Overview

Squire (Sqyre) is a macro builder and automation tool originally built for Dark and Darker, but is expanding to include any program a user wants to add. It provides a GUI for creating and executing complex automation macros using computer vision, OCR, and input simulation

## Core Capabilities

- Visual macro editor with tree-based action organization
- Image recognition and template matching for UI elements
- OCR text detection for dynamic UI reading
- Mouse and keyboard automation
- Hotkey-triggered macro execution
- Multi-program support with program-specific configurations

## Target Use Case

Automates repetitive clicking and screen management tasks in programs by detecting UI elements through computer vision and executing predefined action sequences.
